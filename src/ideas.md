## 아이디어 1: "Sovereign Command" — 군사 지휘 체계 미학

**Design Movement**: Military Command Center / Strategic Operations Aesthetic
**Core Principles**:
1. 정보 계층 구조의 절대적 명확성 — 지휘관이 한눈에 상황을 파악하듯
2. 수직적 권위 — 위에서 아래로 흐르는 명령 체계를 시각적으로 표현
3. 데이터 기반 신뢰 — 숫자와 수치가 전면에 배치되어 즉각적 신뢰 형성

**Color Philosophy**: 딥 네이비(#0a1628~#152e55)는 군함의 철갑, 골드(#d4a827)는 장교 계급장의 금박. 이 조합은 "명령을 내릴 자격이 있는 존재"의 느낌을 전달.

**Layout Paradigm**: 비대칭 2컬럼 레이아웃. 좌측은 내러티브(명령문), 우측은 데이터/카드(보고서). 섹션 간 골드 라인 디바이더로 "명령 하달선" 표현.

**Signature Elements**:
- 골드 그라데이션 텍스트로 핵심 수치 강조
- 반투명 글래스모피즘 카드 (지휘소 모니터 느낌)
- 미묘한 그리드 패턴 배경 (전술 지도 느낌)

**Interaction Philosophy**: 호버 시 카드가 위로 부상하며 골드 보더 활성화 — "선택된 명령"의 느낌

**Animation**: 스크롤 시 섹션이 아래에서 위로 페이드인 — 보고서가 올라오는 느낌. 숫자 카운트업 애니메이션으로 성과 수치 강조.

**Typography System**: Noto Serif KR(헤드라인, 권위) + Inter(본문, 정밀). 헤드라인은 굵고 크게, 본문은 가볍고 절제.

</idea>

## 아이디어 2: "Tidal Cartography" — 해도(海圖) 기반 내비게이션 미학

**Design Movement**: Nautical Cartography / Maritime Wayfinding Aesthetic
**Core Principles**:
1. 항해의 은유 — 각 섹션이 항해 지점(Waypoint)처럼 연결
2. 깊이감 — 바다의 수심처럼 레이어가 겹치는 시각적 깊이
3. 정밀한 경로 — 최적 항로를 찾듯 사용자를 정확한 정보로 안내

**Color Philosophy**: 네이비는 심해의 색, 골드는 나침반과 항해 도구의 황동. "바다를 지배하는 지능"의 메타포.

**Layout Paradigm**: 수평 스크롤 힌트가 있는 풀스크린 섹션. 각 섹션 사이에 파도 형태의 SVG 디바이더. 좌우 비대칭으로 "항해 일지" 느낌.

**Signature Elements**:
- 등고선/해도 스타일의 미묘한 배경 패턴
- 나침반 로즈 모티프의 섹션 구분자
- 파도 형태 SVG 트랜지션

**Interaction Philosophy**: 스크롤이 항해처럼 느껴지도록 — 각 섹션 진입 시 부드러운 패럴랙스

**Animation**: 파도처럼 물결치는 배경 애니메이션, 카드 진입 시 좌우에서 슬라이드인

**Typography System**: Playfair Display(영문 헤드라인) + Noto Serif KR(한글 헤드라인) + Inter(본문). 해도의 캘리그래피 느낌.

</idea>
<text>해양 물류라는 도메인 특성을 시각적으로 반영하되, AX 파트너의 권위를 해도(海圖)의 정밀함으로 표현. 다만 해양 테마가 너무 강하면 범용 AX 파트너 이미지가 약해질 수 있음.</text>

## 아이디어 3: "Institutional Authority" — 제도적 권위 미학

**Design Movement**: Swiss International / Institutional Modernism
**Core Principles**:
1. 절제된 권위 — 과시하지 않으면서도 무게감 있는 디자인
2. 타이포그래피 중심 — 이미지보다 텍스트의 힘으로 설득
3. 그리드의 엄격함 — 모든 요소가 수학적으로 정렬

**Color Philosophy**: 네이비는 정장의 색, 골드는 인장(印章)의 색. "국가 기간 산업을 다루는 기관"의 느낌. 화이트 섹션과 다크 섹션의 극적 대비.

**Layout Paradigm**: 엄격한 12컬럼 그리드. 넓은 여백. 텍스트 블록과 데이터 블록이 교차하는 리포트 스타일. 풀블리드 다크 섹션과 화이트 섹션의 교차.

**Signature Elements**:
- 대형 세리프 타이포그래피 (신문 1면 헤드라인 스타일)
- 골드 언더라인 강조 (공식 문서의 밑줄)
- 미니멀한 아이콘 + 큰 숫자 조합

**Interaction Philosophy**: 최소한의 인터랙션. 호버 효과는 미묘한 색상 변화만. "가만히 있어도 권위가 느껴지는" 정적 아름다움.

**Animation**: 스크롤 시 텍스트만 페이드인. 과도한 모션 배제. 숫자 카운트업만 허용.

**Typography System**: Noto Serif KR 900(헤드라인, 극도로 굵게) + Inter 300(본문, 극도로 가볍게). 극단적 대비로 시각적 긴장감.

</idea>
