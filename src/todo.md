# 매바라 등급 구조 변경 TODO

## 새 등급 체계 (4단계, 이름 통일)
- Basic → Standard → Advanced → Premium
- 각 서비스(AX, Solutions, Academy) 모두 동일 4등급
- 묶음 패키지 시 15~20% 할인

## 등급별 내용 요약 (가격 제외)

### Basic (입문 단계)
- AX: 1-Day 데이터 진단 워크숍 + AI 도입 우선순위 리포트
- Solutions: 표준형 사내 지식 베이스(Brain Lite) 구축 + 매뉴얼 50종 학습
- Academy: 전사 대상 'AI 시대의 생존 전략' 특강 (1회, 2시간)
- 홈페이지 카피: "일단 가볍게 시작하고 싶다면"

### Standard (성장 단계) — Best Value
- AX: 핵심 1개 공정 정밀 분석 + 데이터 파이프라인 설계도 + AX 로드맵
- Solutions: 업무 대행 에이전트(Pilot) 엔진 개발 + 메신저(Slack/Teams) 연동
- Academy: 실무자 10인 대상 4주 집중 AI 부트캠프 (주 1회 대면 코칭)
- 홈페이지 카피: "눈에 보이는 성과를 즉시 내고 싶다면"

### Advanced (혁신 단계)
- AX: 전사적 AI-Native 성숙도 진단 + AI 보안 거버넌스 수립 + 인증 심사
- Solutions: 처방형 분석 대시보드 구축 + 기업 맞춤형 최적화 알고리즘 개발
- Academy: 핵심 인재 5인 대상 'AI 나노디그리' 인증 과정 + 부원장 직접 멘토링
- 홈페이지 카피: "기업의 DNA를 완전히 바꾸고 싶다면"

### Premium (지배 단계)
- AX: 글로벌 표준 AI 아키텍처 설계 + 그룹사 단위 마스터 플랜
- Solutions: 매바라 One(항만/물류 특화 OS) 전체 이식 + 현장 실시간 동기화
- Academy: C-Suite 전담 기술 코칭(월 2회) + 글로벌 AI 기술 트렌드 리포트
- 홈페이지 카피: "산업의 압도적 1위를 원하신다면"
